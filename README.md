# 📊 Bank Transaction Dashboard

## 📌 Overview
This project analyzes banking transaction data using SQL and Power BI to uncover insights about transactions, customers, and investment behavior.

---

## 🛠 Tools Used
- MySQL (Data storage & analysis)
- Power BI (Dashboard & visualization)
- Excel (Dataset)

---

## 📂 Project Workflow
1. Imported CSV/Excel data into MySQL
2. Cleaned and transformed data (date formatting, structure)
3. Performed SQL queries (aggregation, grouping, filtering)
4. Built an interactive dashboard in Power BI

---

## 📈 Dashboard Features
- Transaction Amount by Account Type
- Transaction Count by Account Type
- Investment Distribution by Type
- Monthly Transaction Trend
- Top 5 Customers by Transaction Amount

---

## 🔍 Key Insights
- Savings accounts slightly lead in total transaction amount
- Transactions are evenly distributed across account types
- Investment preferences are balanced across types
- Monthly trends show stable fluctuations
- Top customers contribute significantly to total transactions
